Download the cMonkey code and data

These downloads contain the R implementation of the version of cMonkey and data described in the manuscript.

*NOTE* a much simpler and easier to use version of cMonkey is implemented as an R library and can be accessed here.

To download the files for the original version of cMonkey, right-click the links below and select "Save link as..."

  • The README file containing a basic description and documentation of this release
  • The RData file containing the R cMonkey implementation
  • The R script file required for running the procedure
  • Organism-specific data files (in RData file format used by code above; contains all relevant expression, sequence and association/network data):
    • H. pylori data file
    • E. coli data file
    • S. cerevisiae data file
    • Halobacterium data
  • All of the above files in a compressed tar archive
  • Organism-specific expression (ONLY) data files (in compressed TSV file format):
    • H. pylori expression data file
    • E. coli expression data file
    • S. cerevisiae expression data file
    • Halobacterium expression data
